# Amazon-Clone-Project
An Amazon Clone developed using HTML and CSS that recreates the user interface of Amazon's homepage. The project focuses on building a responsive layout, navigation bar, hero section, product cards, and footer while improving skills in HTML structure, CSS Flexbox, Grid, and responsive web design.

# Features
Amazon-inspired homepage layout
Navigation bar with search box
Hero banner section
Product cards and categories
Footer with multiple sections
Clean and organized code structure
Technologies Used
HTML5
CSS

# Learning Outcomes
Improved HTML semantic structuring
Better understanding of CSS Flexbox and Grid
Responsive web design principles
Frontend UI development best practices
